# AI Website Copy Generator: Rajwadi Chai ☕

## 📌 Project Overview
This repository contains a structured prompt engineering framework designed to generate high-conversion, professional website copy for local businesses. 

Most local businesses suffer from weak, generic website copy. This project solves that by using targeted, role-based AI prompts to generate clear, benefit-driven, and highly persuasive content that is ready to be published via tools like Framer AI or Lovable.dev.

## 🏢 Chosen Business Context (Client Profile)
For this project, I engineered a complete website copy suite for a real local business:
- **Business Name:** Rajwadi Chai
- **Location:** Rashtriya Raksha University (RRU) Lavad, Gandhinagar
- **Target Audience:** University students, local professionals, and weekend visitors looking for a relaxed, aesthetic spot to study or hang out.
- **Core Offerings:** Signature Rajwadi thick tea, Kullad Kesari Ukala, Classic Chais, Kullad Coffee, Bun Muska, and Indori Poha.
- **Primary Goal:** Drive foot traffic by encouraging visitors to view the menu and get directions.

## 🧠 The Prompt Logic
This system avoids "random AI text" by using a **Modular Prompting Strategy**:

1. **The Master Context Prompt (System Prompt):** Sets the AI's persona as a "world-class conversion copywriter." It injects the specific client profile, defines the warm/energetic tone, and explicitly forbids generic AI jargon (like "unleash" or "elevate").
2. **Homepage Prompt:** Focuses strictly on the above-the-fold content, demanding a clear value proposition (H1) and a trust-building introductory snippet.
3. **Menu/Services Prompt:** Structures the core offerings by forcing the AI to provide a description *and* a specific customer benefit for each item.
4. **Call-to-Action (CTA) Prompt:** Generates distinct, goal-oriented CTAs (Menu-focused, Location-focused, Vibe-focused) to place throughout the site.

## 🛠 Tools Used
- **Prompt Engineering & Generation:** Google Gemini / ChatGPT
- **Intended Deployment:** Framer AI / Lovable.dev (The generated copy is formatted for immediate use in these visual builders).

## 📂 Repository Structure
- `prompts.md` - Contains the exact, reusable prompt templates designed for this project.
- `generated_copy.md` - Contains the final, publish-ready website copy generated by the AI.
